일정 기간동안 진행한 개발 기록을 묶어 dev.log라는 카테고리에 적어보려고 한다. 아마 당분간은 블로그 개발에 대한 글을 주로 올릴 것 같다.

블로그 개발

지금껏 개발을 할 때, 원하는 기능을 구현하기 위해서는 라이브러리를 찾고 적용하는 데 많은 시간을 썼었다. 그러다 보니 라이브러리에 의존을 많이 했고 내부 구현사항을 모르는 상태에서는 에러 원인 파악이나 기능 구현에 제한이 있기도 했었다.

또한 다른 개발자들이 만들어놓은 결과물로만 조합해서 개발하다 보니 어느 순간부터 여러 라이브러리가 섞여 결국 프로그램이 어떻게 동작하는지를 잘 모르고 있었다.

무언가를 이해하려면 직접 만들어보는 게 좋은 방법이므로 이번 블로그 개발은 그동안 당연하게 사용해왔던 기술의 의존성을 줄이고 무엇이, 왜, 언제 필요한지에 대해 느껴보고 직접 만들어보면서 어떻게 동작할지에 대해 생각해보고 알아가보려고 한다.

  • 이미지 최적화

  • 방문자 통계, 분석

  • Key Value 데이터베이스 개발
